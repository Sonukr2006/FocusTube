# Local Sounds Folder

Keep all SoftMurmure audio files in this folder:

- `client/public/sounds/rain.mp3`
- `client/public/sounds/thunder.mp3`
- `client/public/sounds/wind.mp3`
- `client/public/sounds/ocean.mp3`
- `client/public/sounds/stream.mp3`
- `client/public/sounds/forest.mp3`
- `client/public/sounds/white-noise.mp3`
- `client/public/sounds/brown-noise.mp3`


Notes:

- Use loop-ready files (seamless start/end) to avoid clicks.
- Preferred format: `mp3` (or `ogg` if you add support later).
- Keep each file reasonably small (for example 1-5 MB each).

In browser these will be available at:

- `/sounds/rain.mp3`
- `/sounds/thunder.mp3`
- ...
